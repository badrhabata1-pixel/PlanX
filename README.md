# 🚀 PlanX | AI-Powered Exam Scheduling System

**PlanX** is an advanced automated system designed to solve the complex problem of university exam scheduling using Artificial Intelligence. It balances student enrollments, room capacities, and time constraints to generate an optimized, conflict-free schedule in seconds.

---

## ✨ Key Features
*   **Smart Data Ingestion:** Automatically identifies and maps columns from various file formats (XLS, XLSX, CSV).
*   **AI Optimization Engine:** Handles complex constraints (student conflicts, room availability, and period gaps).
*   **Real-time Processing:** Provides a live console view of the AI engine's progress.
*   **Automated Export:** Generates the final optimized schedule directly into a professional Excel file.
*   **Multi-language Support:** Fully responsive web interface with Arabic and English support.

---

## 🛠️ Tech Stack
*   **Backend Engine:** Python 3.13 (Pandas, Openpyxl, Datetime).
*   **Web Interface:** PHP 8.x, HTML5, CSS3 (Modern Glassmorphism UI).
*   **Database:** MySQL (for system configurations).
*   **Version Control:** Git & GitHub.

---

## 📁 Project Structure
*   `/planx_ai_engine`: The core AI optimization logic and data processing modules.
*   `/data/raw`: Storage for ingested student and room data.
*   `process.php`: The bridge between the web UI and the Python engine.
*   `workspace.php`: The user dashboard for data uploading and protocol initiation.

---

## 🚀 How it Works
1. **Ingestion:** Upload student rosters and room constraints.
2. **Stabilization:** The AI Engine cleans the data and builds a conflict matrix.
3. **Optimization:** The engine runs optimization algorithms to find the best time-slots.
4. **Delivery:** Download the final optimized Excel schedule.
